This folder contains .pkt files, these are text files are required to show a map in the multiplayer map list, they also control the listed name.

The following 3 .pkt files are placed here on a fresh copy (but more can be added later):

ra1originalmaps.pkt - A list of the original Red Alert 1 maps (without expansion(s))
sc-AMmaps.pkt - Contains a list of Aftermath maps, note that /optional/sc-AMmaps.mix also needs to be placed in the Red Alert 1 folder
sc-CSmaps.pkt - Contains a list of Counterstrike maps, note that /optional/sc-CSmaps.mix also needs to be placed in the Red Alert 1 folder
